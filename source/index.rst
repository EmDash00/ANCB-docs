Welcome to ANCB's documentation!
================================================================

Another Numpy Circular Buffer is an efficient, fast, and powerful NumPy 
compatible circular buffers for use in data processing, especially 
real-time data processing.

Installation
------------
At the command line::

   $ pip3 install ancb
